Nasjonal arkitektur for samhandling

Denne prototypen tester en enkel struktur for a forklare hva nasjonal arkitektur er, hvilke kapabiliteter og prinsipper som inngar, og hvilke felles ressurser som finnes i okosystemet.

Hva handler dette om?

Nasjonal arkitektur beskriver felles evner, prinsipper og ressurser som gjor det mulig a utvikle sammenhengende digitale tjenester pa tvers av virksomheter.

Maalet med denne testflaten er a gi en tydelig inngang for tre brukerbehov:

  • forsta modellens kjernebegreper raskt
  • finne sammenhengen mellom prinsipper og kapabiliteter
  • finne konkrete produkter som understotter kapabilitetene

Hvordan lese modellen

  • Kapabiliteter viser hvilke evner okosystemet trenger.
  • Prinsipper viser hvilke arkitekturfaglige foringer som bor styre utviklingen.
  • Ressursoversikt viser produkter og felleslosninger som bidrar til a realisere kapabilitetene.

Foreslatt lesevei

  1. Start pa kapabiliteter for a fa oversikt over evnene i modellen.
  2. Les prinsipper for a se hvilke arkitekturfaglige foringer som styrer prioriteringer.
  3. Gaa til ressursoversikt for a se hvilke produkter som bidrar i praksis.

Kapabiliteter

Kapabiliteter Kapabilitetene beskriver hvilke evner som trengs for a sikre samhandling, gjenbruk og strategisk retning i et nasjonalt digitalt okosystem. Hovedkapabiliteter Strategisk styring Evnen til a sette retning for nasjonal arkitektur og realisere strategiske mal. Delkapabiliteter: Finansiering Arkitekturstyring Samordning Samarbeid Evne til samarbeid og samhandling pa tvers av offentlig og privat forvaltning. Delkapabiliteter: Organisatorisk samhandling Samarbeidsarenaer og nettverk Tjenesteforvaltning Sluttbrukertjenester Evne til a tilby en sammenhengende digital brukeropplevelse gjennom et okosystem av standardiserte og integrerbare tjenester.

Prinsipper

Prinsipper Prinsippene gir faglige foringer for hvordan nasjonal arkitektur bor utvikles og forvaltes. Ressursoversikt

Ressursoversikt Ressursoversikten samler produktbeskrivelser for nasjonale fellesløsninger og relaterte ressurser. Hva skal denne siden gi? Denne siden er inngangen til produktoversiktene i results/Produktbeskrivelser/ og hjelper brukeren å se hvilke produkter som støtter hvilke kapabiliteter. I denne prototypen bruker vi kun siste versjon per produkt. Produkter Alle produkter er samlet i én tabell: Gå til produktsamling Hver rad viser: produktnavn siste versjon og forfatter kort beskrivelse direkte lenke til markdownfil i GitHub Regel for versjonsvalg Vi velger alltid høyeste versjonsnummer per produktnummer.

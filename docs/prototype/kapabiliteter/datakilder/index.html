Datakilder

Datakilder

Evne til å tilgjengeliggjøre og forvalte data som en nasjonal fellesressurs, slik at de kan oppdages, forstås og gjenbrukes på en sikker og standardisert måte.

Delkapabiliteter

DelkapabilitetBeskrivelseProdukter
GrunndataEvne til å identifisere og formelt anerkjenne autoritative datakilder basert på en systematisk vurdering av deres verdi.0
SanntidsdataEvne til å sikkert prosessere og distribuere kontinuerlige datastrømmer for å muliggjøre sanntidsinnsikt, automatisering og umiddelbar respons.0
Åpne dataEvne til å tilgjengeliggjøre åpne data slik at den kan leses og tolkes av både maskiner og mennesker, og som alle kan få tilgang til, bruke og dele.0
TestdataEvne til å generere og tilgjengeliggjøre representative og personvernsikre datasett.0
Ustrukturerte dataEvne til å tilgjengeliggjøre og forvalte data som ikke er påført en forhåndsdefinert struktur.0

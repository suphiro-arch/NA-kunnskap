# Kunnskapsgrunnlag - dataspaces

## Ressurs ID
DIGDIR-040

## Ressurskategori
Normerende ressurs

## Type normerende ressurs
Kunnskapsgrunnlag

## Status/Livsfase
**Under utvikling (paabegynt)** - kortutkast for testing av visning pa nett og bruk i analyser.

## Kort beskrivelse
Forelopig kunnskapsgrunnlag om deltakelse i europeiske dataomrader og relevante konsekvenser.

## Formaal og normerende rolle
Ressursen er opprettet som en kort paabegynt beskrivelse for a kunne testes i webvisning og i analysearbeid. Innholdet skal utdypes med tydelig scope, rolle og forvaltningsmodell i neste iterasjon.

## Kapabiliteter
- Samarbeid: Samarbeidsarenaer og nettverk
- Datautveksling og integrasjon: Dele data med andre

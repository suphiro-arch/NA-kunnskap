# Produkt-canvas: data.altinn.no

Målgruppe: Hovedfokus er forretningssiden og strategisk arkitektur.

## Navn
data.altinn.no

## Ressurs ID
17 (Produktliste NA-kunnskap).

## Status/Livsfase
Produksjon som Altinns dataportal for deling og oppdagelse av dataressurser.

## Modenhet
Middels til høy:
- Etablert portal med aktiv funksjon i Altinn-økosystemet.
- Understøtter datadeling i samspill med Altinns øvrige produkter.
- Modenhet påvirkes av løpende utvikling i plattform og datadelingstjenester.

## Kort beskrivelse
data.altinn.no er Altinns dataportal for publisering og oppdagelse av dataressurser knyttet til Altinn-økosystemet. Løsningen gjør det enklere å finne relevante data og grensesnitt for videre bruk i tjenester, integrasjoner og analyser.

## Kapabiliteter
- Informasjonsforvaltning: Oversikt over datasett
- Informasjonsforvaltning: Oversikt over API
- Datautveksling og integrasjon: Dele data med andre
- Datautveksling og integrasjon: Bruke data fra andre
- Tjenesteutvikling: Integrerbare tjenester
- Standardisering: Forvaltningsstandarder

Grunnlag: kapabiliteter mappet mot `arkitektur/kapabiliteter/capabilities.yaml`.

## Produktmål
- Øke synlighet og gjenbruk av dataressurser i Altinn-domenet.
- Forenkle integrasjon mot Altinn-relaterte datakilder og API-er.
- Understøtte mer effektiv tjenesteutvikling og samhandling.

## Brukerbehov
- Tjenesteeiere trenger kanal for å gjøre dataressurser oppdagbare.
- Integrasjonsteam trenger rask tilgang til metadata og inngang til API-er/datakilder.
- Arkitekturmiljøer trenger oversikt over tilgjengelige ressurser før nyutvikling.

## Hvem er brukerne og brukersegmentene
- Offentlige virksomheter i Altinn-økosystemet.
- Utviklere, arkitekter og integrasjonsteam.
- Leverandører som bygger løsninger basert på Altinn-data og grensesnitt.

## Hovedfunksjoner
- Oppdagelse av dataressurser og API-relatert informasjon i Altinn-kontekst.
- Metadataorientert oversikt over tilgjengelige ressurser.
- Kobling mellom datadeling og øvrige Altinn-produkter.

### Scope og avgrensning
- Inngår:
  - portal for metadata og oppdagbarhet
  - støtte til gjenbruk og integrasjonsforberedelse
- Inngår ikke:
  - full dataplattformdrift hos publiserende virksomheter
  - all dataforvaltning utenfor Altinn-domene

## Veikart over kommende funksjonalitet
- Forventet løpende forbedring av metadata-kvalitet og brukerreise.
- Videreutvikling i takt med Altinns datadelings- og API-strategi.
- Konkrete prioriteringer må verifiseres mot oppdatert produktinformasjon.

## Forretningsverdi/Verdiforslag
- Reduserer friksjon ved oppdagelse av relevante data i Altinn-økosystemet.
- Øker sannsynlighet for gjenbruk fremfor ny etablering av parallelle løsninger.
- Forbedrer beslutningsgrunnlag i arkitektur- og integrasjonsarbeid.

## Utfordringer og risiko
- Juridisk risiko: utydelige bruksvilkår/rettigheter kan gi feil bruk av data.
- Teknisk risiko: varierende metadata-kompletthet kan gi lav oppdagbarhet.
- Sikkerhetsrisiko: feil klassifisering kan gi utilsiktet informasjonseksponering.
- Leverandørrisiko: ulik kvalitet i publisering mellom aktører.
- Brukerrisiko: brukere kan tolke metadata feil uten tilstrekkelig kontekst.

# Produktbeskrivelser

Denne mappa samler produktbeskrivelsene for produktområdet i repoet.

- Filene er del av det kuraterte arkitekturgrunnlaget, ikke en generell leveransemappe.
- `produktnummerering.md` i mappa over er register- og oversiktsfila for samme område, og er operativ master for produktregisteret.
- Webgeneratorer og koblingsfiler skal peke hit når de trenger produktbeskrivelser.
